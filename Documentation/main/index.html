Introduction

MAGIC ecommerce is the best solution to make your own costume modern ecommerce website ..easy to use .. fast to install. This documentation will guide you through installing the CMS and exploring the various components that are bundled with the CMS.

Installation Guide

* When you login to the first time into your CMS, you will be the main admin of website and you can add more admins from control panel > users > Edit/Delete .

  1. First of all, Download the script and extract it on your device.
  2. Open your phpMyAdmin and create new database and call it magic.
  3. Import project.sql to your new database that you created, you can find database.sql from magic CMS folder that you extracted.
  4. upload the folder to your host.
  5. open http://[your_project_url]/install and follow the steps.
  6. Login to username: admin and password: admin123.
  7. Enjoy it.

Get protocols

In any page other than cart, order & item you can use cetain get protocols to search within displayed items:

Get value Description
?search= Search for items that make there title, discription or tags
?fav= Search for fav if given value is 1 for example: ?fav=1
?product_catigory= Search for given product catigory
?vendor= Search for given vendors
?product_type= Search for given product_type
?collection= Search for given collection
?barcode= Search for given barcode

Browser Support

Note!

IE10 does not support transitions or animations. The template will function properly but it won't use animations/transitions on IE10.

Crypto Tokenizer Admin supports the following browsers:

  • IE11+
  • Edge (latest)
  • Firefox (latest)
  • Safari (latest)
  • Chrome (latest)
  • Opera (latest)
